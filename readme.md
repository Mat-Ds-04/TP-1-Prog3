# Trabajo Práctico Progr. 3. Primera pagina Web con HTML y CSS

Este repositorio contiene el primer proyecto de pagina Web. Trabajo de Programación.

## Publicación de la Página.

La página del proyecto se encuentra aquí: <target _blank> https://mat-ds-04.github.io/TP-1-Prog3/

# Estructurización.

La pagina consiste en 3 html enlazados a través de links integrados a la misma.

1. Un index.html, como pagina principal.

2. Un horario.html, como cronograma del evento.

3. Una receta.html, como plan de almuerzo del dia.


# Lenguajes utilizados.

* HTML5. Su función es la estructura y el funcionamiento de la página.

* CSS. Su función es la estilización de la página.
